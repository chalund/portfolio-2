## Getting started

### Installation

1. Clone the project

```
https://github.com/chalund/portfolio-2.git
```

2. Install NPM packages

```
npm install or yarn install
```

3. Start the project

```
npm start
```

### Run the application

1. Start the development server

```
npm run dev
```

2. View in Browser
   Open the provided localhost link in your preferred web browser
